# Running PA19

* Web interface shows Compute AND, Run All option.

### Run All option
##### For Alice
- First send request to /pa14/demo_setup
- Next send request to /pa15/keygen
- Next send request to /pa16/keygen
#### For Bob
- First send request to /pa14/demo_setup
- Next send request to /pa15/keygen
- Next send request to /pa16/keygen

* Finally executes /pa19/run_all api endpoint.
